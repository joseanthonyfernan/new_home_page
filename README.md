# ModernOne Web Application

A premium, responsive one-page web application built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern Aesthetics**: Sleek white, red, and blue color palette with glassmorphism and smooth animations.
- **Top Navigation**: Sticky header with smooth scroll links.
- **SEO Optimized**: Semantic HTML5 structure and meta tags included.
- **Interactive**: Scroll-triggered animations and a functional contact form simulation.

## Tech Stack
- **Structure**: HTML5
- **Styling**: Bootstrap 5 + Vanilla CSS
- **Interactivity**: Vanilla JavaScript
- **Animations**: Animate.css
- **Typography**: Outfit (Google Fonts)
- **Icons**: Bootstrap Icons

## How to Run
Simply open `index.html` in your web browser. Or, if you have XAMPP running, move the folder to your `htdocs` and access it via `http://localhost/new_home_page/`.
